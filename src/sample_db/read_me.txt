This module is used to generate sample DB, contain responses from getter API ,such as get_list_posts, get_saved_search, ....
=> help for coding UI without real APIs .


Use : 
	+ Open : 'https://repl.it/repls/SimpleSuperbMice' 
	+ copy files : index.js (execute file) ,package.js ( contain libraries) and videos ( contain sample videos ) from this module to 
		this website (run on NodeJs)
	+ Run index.js to generate DB on sample_db.json
	+ file : sample_db_structure.js describe structure of sample DB and also type, optional, use of fields on objects
